HR-Analytics - Predict-Employee Attrition :
- This project analyzes employee attrition using exploratory data analysis, machine learning, and data visualization. A Logistic Regression model was trained to predict attrition risk with approximately 88% accuracy, while an interactive Power BI dashboard highlights key attrition drivers such as overtime, job satisfaction, income, and department-level trends. The project also includes data-driven attrition prevention recommendations for HR decision-making.
